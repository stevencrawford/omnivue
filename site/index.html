Features

Everything you need to review, search, and organize your AI coding sessions.

Unified View

Browse conversations, plans, and file diffs from all your agents in a single tabbed interface. Normalized across OpenCode, Copilot, and Cursor.

Full-Text Search

Search across every message, tool call, plan, and diff. FTS5-powered with instant results. Scope to a single session or search everything.

User Folders

Organize sessions into virtual folders with nesting, colors, and icons. Group by project, feature, or anything you like.

Scratch Notes

Attach rich text or code notes to any session. Use the WYSIWYG editor or Monaco code editor. Perfect for review notes, summaries, and annotations.

Resume & Share

One-click copy of the resume command for any session. Share deep links via URL hash — #/session/{id}/step/{n}.

Read-Only Safe

Never modifies agent databases. All connections use read-only mode, verified at open time. Your agent data stays untouched.

Quick Start

Install, initialize, and start browsing in three commands.

Terminal 
$ curl -LO https://github.com/stevencrawford/sess/releases/latest/download/sess_$(uname -s)_$(uname -m).tar.gz
$ tar xzf sess_*.tar.gz && sudo mv sess /usr/local/bin/

$ sess init
  sess: scanning for AI agent sessions...
  sess: found 2 source(s):

  [opencode] /Users/user/.local/share/opencode (14 sessions)
  Add this source? [Y/n] Y
  Added.

  [copilot] /Users/user/.copilot (8 sessions)
  Add this source? [Y/n] Y
  Added.

$ sess
  sess: server started at http://localhost:6275 (pid 9876)

Keyboard-Driven

Navigate sessions with jk, switch tabs with ⌘1⌘2, and search with ⌘P. Built for power users who live in the terminal.

KeyAction
j / ↓Next session
k / ↑Previous session
⌘1Conversation tab
⌘2Diff tab
⌘P / ⌘KOpen search
Single Binary

Go backend with an embedded React SPA. Download one binary, run it, and go. No Node.js, no database servers, no runtime dependencies.
