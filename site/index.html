Multi-Agent Session Manager

View, search, and manage all your AI coding agent sessions in one place.

OpenCode Copilot Cursor Pi Codex
Get Started Download
Unified View

Every session,
one browser

See all your AI coding sessions from every agent in a single tree. Sessions are grouped by repository with color-coded agent badges — OpenCode, Copilot, Cursor, Pi, and Codex.

  • Color-coded agent badges — instantly identify the source
  • Sessions grouped by repository — context at a glance
  • Resizable sidebar with persisted width
  • Live updates via SSE — new sessions appear in real time
Conversation View

Read every message,
every tool call

Full conversation history with alternating user/assistant messages, tool call outputs, and collapsible reasoning blocks. Normalized across all agents.

  • Collapsible reasoning/thinking blocks
  • Dedicated renderers for each tool type (bash, edit, read, grep, glob, delete, task, question, ...)
  • Markdown rendering with syntax-highlighted code blocks
  • Step events show plan-mode to code-mode transitions
File Diffs

Review every change,
side by side

See exactly what files were created, modified, or deleted in each session. Expand any file to view the full unified diff with syntax-highlighted additions and deletions.

  • File change list with addition/deletion counts
  • Expandable unified diffs with green/red line highlights
  • Raw edit/write tool call data for precise before/after
  • File status icons — added, modified, deleted, renamed
Full-Text Search

Search everything,
instantly

FTS5-powered full-text search across every message, tool call, plan, and diff. Scope to a single session or search all your agents at once.

  • Incremental indexing with content-hash deduplication
  • Inline search typeahead — ⌘P
  • Full-screen results drawer with highlighted snippets
  • Scope to session or search globally
User Folders

Organize by project,
not by agent

Create virtual folders to organize sessions from all agents into projects. Sessions from OpenCode, Copilot, Cursor, Pi, and Codex all live side by side.

  • Nested folders with custom colors and icons
  • Assign any session to any folder — regardless of agent
  • Project-based browsing alongside the session tree
  • Persisted in the sess database — survives restarts

Scratch Notes

Per-session rich text or code notes. WYSIWYG (TipTap) or Monaco editor. Auto-save, perfect for review notes and annotations.

Resume & Share

One-click copy of the resume command. Deep-link any step with #/session/{id}/step/{n} for sharing.

Read-Only Safe

Never writes to agent databases. All connections use ?mode=ro — verified at open time. Your agent data stays untouched.

Quick Start

Install, initialize, and start browsing in three commands.

Terminal 
$ curl -LO https://github.com/stevencrawford/sess/releases/latest/download/sess_$(uname -s)_$(uname -m).tar.gz
$ tar xzf sess_*.tar.gz && sudo mv sess /usr/local/bin/

$ sess init
  sess: scanning for AI agent sessions...
