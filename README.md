### Installation

```bash
# From source (requires Go 1.26+ and pnpm)
make build

# Binary
curl -fsSL https://github.com/stevencrawford/omnivue/releases/latest/download/omnivue_darwin_arm64.tar.gz | tar xz
```

### Quick Start

```console
$ omnivue init
$ omnivue
```
